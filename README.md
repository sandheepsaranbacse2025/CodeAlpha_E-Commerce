# CodeAlpha E-Commerce

A modern E-Commerce web application developed as part of the CodeAlpha Internship Program. The platform provides a seamless online shopping experience with product browsing, cart management, user authentication, and order processing features.

## Overview

This project is designed to simulate a real-world online shopping platform where users can explore products, manage their shopping cart, and complete purchases through an intuitive interface.

## Features

* User Registration and Login
* Product Catalog Management
* Product Search and Filtering
* Shopping Cart Functionality
* Order Management
* Responsive User Interface
* Secure Data Handling
* Admin Management Features

## Installation

1. Clone the repository

```bash
git clone https://github.com/sandheepsaranbacse2025/CodeAlpha_E-Commerce.git
```

2. Navigate to the project directory

```bash
cd CodeAlpha_E-Commerce
```

3. Configure your local server environment (XAMPP/WAMP/LAMP).

4. Import the database file into MySQL.

5. Update database credentials in the configuration file.

6. Start Apache and MySQL services.

7. Open the application in your browser.

## Usage

* Register a new account or log in.
* Browse available products.
* Add products to the cart.
* Manage cart items.
* Complete the checkout process.
* View and manage orders.
